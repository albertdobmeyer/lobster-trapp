What It Does

Bootstrap Detects prerequisites (Podman/Docker), clones submodules, builds containers — guided setup wizard for non-technical users.
Dashboard Start, stop, and monitor components through a manifest-driven interface. Every button, command, and status badge is declared in component.yml, not hardcoded.
Security Surface Verify results, inspect proxy logs, review scan findings. The reason you need this instead of a terminal.

Architecture

The Tauri backend contains zero knowledge of what's inside any component. Each component self-describes via component.yml. Drop in any component with a valid manifest and it renders correctly. The schema is the contract. The GUI is generic. The components are independent.
